-----------------------------------------------

    * Target MCU Bootloader firmware template source
    * Target MCU Application firmware template source 
    - ReleaseNotes.txt (this file)

Note:
The PC software "Silicon Labs Serial Bootloader Data Source" source code (Visual C#) 
and installer (setup.exe) are included as part of AN533SW.zip. This software is required for 
this bootloader framework. To install, run the following file:
...\DataSource_Software\Release\setup.exe
